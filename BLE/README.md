# BLE CTF

**Это приложение было создано специально для проведения BLE CTF!**

*В случае успешного прохождения CTF на экране Flipper Zero появится QR-код для присоединения к приватному чату!*

## Задание
1. Каждую секунду приложение посредством BLE шлёт зашифрованный текст под определённым UUID характеристики
2. **UUID характеристики:** `19ed82ae-ed21-4c9d-4145-228e62fe0000`
3. Получите и расшифруйте полученный текст, чтобы понять, что делать дальше.
